**Project Description**
This project is a simple Tic Tac Toe (X-O) game developed using C# and Windows Forms.
It demonstrates basic game logic, event handling, and user interface interaction in a desktop application.

---

**Features**

* Two-player X-O game
* Image-based user interface using PictureBox
* Automatic win detection (rows, columns, diagonals)
* The game stops automatically when a player wins
* Simple and beginner-friendly design

---

**Technologies Used**

* C#
* Windows Forms (WinForms)
* Visual Studio
* .NET Framework

---

**How to Run the Project**

1. Download or clone the project from GitHub
2. Open the solution file using Visual Studio
3. Run the project by clicking the Start button
4. The game window will appear and you can start playing
